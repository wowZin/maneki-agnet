# Level 1: Agent Room + One Agent

Use this level when you have one Hermes agent and want a side control plane.

You create:

```text
agents/hermes-life/
  inventory.md
  docker.md
  env-map.md
  runbook.md
  backup.md
```

No orchestrator. No task bus. No specialist team.

Goal: one clean source of truth for the agent.
