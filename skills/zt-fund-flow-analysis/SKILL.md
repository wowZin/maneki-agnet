# ZT Fund Flow Analysis

基于5维度资金面分析框架，量化评估个股短期涨停爆发概率。仅做资金面逻辑研判，不参与基本面、技术形态、情绪题材分析。

This skill runs on the **zt-fund-flow** agent.

## 数据源

Tushare Pro MCP (已配置在 Hermes Agent config.yaml)

使用的 MCP 工具:
- `moneyflow`: 个股资金流向 (大单/中单/小单买入卖出金额)
- `moneyflow_dc`: 东方财富个股资金流向 (主力/超大/大/中/小单净流入)
- `moneyflow_hsgt`: 沪深港通资金流向 (北向资金总量)
- `hsgt_top10`: 沪深港通十大成交股 (北向资金个股明细)
- `top10_floating_holders`: 十大流通股东持仓变动 (机构/社保/公募增减仓)

## 核心分析规则（固定执行逻辑）

- **优先级规则**：超大单主力净流入 > 龙虎榜游资机构入驻 > 分时资金持续抢筹 > 北向资金增持 > 尾盘资金回流，多重资金信号叠加判定为高潜力涨停标的。
- **避雷优先规则**：出现主力资金连续净流出、超大单砸盘、龙虎榜资金出逃、主力减仓、分时持续背离，直接降低涨停评级，严重者直接剔除潜力标的池。
- **判定标准**：单一资金信号仅为小幅加分，主力大幅净流入 + 游资机构抱团 + 分时强回流 + 北向持续加仓 + 无大额抛压多维度共振，判定为高强度资金驱动涨停潜力股。
- **输出要求**：每次分析需明确标注【涨停预测（是/否）】+【核心利好资金逻辑】+【资金面风险隐患】。

## Steps

1. **Receive task from team-leader**
   - Read task from inbox: `/srv/agent-bus/zt-fund-flow/inbox/`
   - Extract stock code, name, board type, and context.

2. **Fetch fund flow data**
   - 调用 MCP 工具 `moneyflow`，获取近5日个股资金流向 (大单小单成交)
   - 调用 MCP 工具 `moneyflow_dc`，获取东方财富资金流向 (主力/超大/大单净流入)
   - 调用 MCP 工具 `moneyflow_hsgt`，获取沪深港通整体资金流向
   - 调用 MCP 工具 `hsgt_top10`，获取北向资金十大成交股 (判断该股是否被北向资金买入)
   - 调用 MCP 工具 `top10_floating_holders`，获取十大流通股东持仓变动

3. **维度1：主力大单资金分析（涨停核心驱动力）** — 权重最高

   研判当日与近3日超大单、大单净流入/流出情况，识别真实主力建仓、抢筹行为。涨停启动必须依托真实大额资金买入，持续大单流入是短线起爆最核心的资金依据。

   3.1 **主力净流入判定**
   - 主力 = 超大单 + 大单
   - 主力净流入 = (超大单买入 + 大单买入) - (超大单卖出 + 大单卖出)
   - 主力净流入占比 > 5%: 强主力抢筹
   - 主力净流入占比 2-5%: 中等主力流入
   - 主力净流入占比 0-2%: 微弱流入
   - 主力净流出: 出货信号

   3.2 **主力流入持续性**
   - 近3日连续净流入: 持续建仓/抢筹
   - 近3日2日净流入1日流出: 波动性流入
   - 近3日仅1日净流入: 单日异动，可靠性低
   - 近3日连续净流出: 持续出货

   3.3 **超大单/大单结构**
   - 超大单占比 > 总成交30%: 大资金主导
   - 超大单买入占比 > 超大单卖出占比70%: 方向性买入
   - 大单与超大单同步流入: 主力一致性高

   3.4 **主力资金评分**
   - 主力净流入>5%+连续3日流入(+25): 最强主力信号
   - 主力净流入2-5%+连续流入(+18): 较强主力
   - 主力微弱流入(+8): 一般
   - 主力净流出(-10): 出货
   - 连续3日净流出+超大单砸盘(-20): 强出货信号

   评分区间：[-20, +25]

4. **维度2：龙虎榜机构游资分析（强势溢价信号）**

   研判个股是否上榜、机构净买入、知名游资介入情况。机构+游资联手扫货的标的，隔日溢价、连板、涨停概率最高；纯散户交易无主力入驻标的无爆发潜力。

   4.1 **龙虎榜上榜情况**
   - 从 moneyflow_dc / top10_floating_holders 间接推断
   - 当日涨幅 > 7% 触发上榜条件
   - 上榜 + 机构净买入: 最强信号
   - 上榜 + 知名游资买入: 强信号
   - 上榜 + 机构卖出: 利空
   - 未上榜: 无龙虎榜数据参考

   4.2 **机构买入判定**
   - 机构净买入占比 > 龙虎榜总买入50%: 强机构支撑
   - 机构净买入占比 20-50%: 有机构参与
   - 机构净卖出: 机构不看好

   4.3 **游资合力判定**
   - 多家知名游资同时买入(>=3席): 游资合力
   - 1-2家游资买入: 有限游资
   - 知名游资卖出: 游资出逃
   - 无游资参与: 缺乏短线催化

   4.4 **龙虎榜评分**
   - 上榜+机构大额净买入+多家游资合力(+25): 最强资金信号
   - 上榜+机构净买入+游资买入(+18): 强合力
   - 上榜+仅游资买入(+12): 短线有催化
   - 上榜+机构净卖出(-10): 机构不认可
   - 上榜+游资出逃+机构出逃(-20): 大资金撤离

   评分区间：[-20, +25]

5. **维度3：北向与公募资金分析（中线资金背书）**

   研判北向资金、QFII、公募基金当日及阶段性增减仓。北向持续加仓代表外部主流资金认可，能够锁住筹码，避免冲高回落，为涨停持续性提供资金兜底。

   5.1 **北向资金持仓变动**
   - 从 hsgt_top10 获取该股北向资金明细
   - 北向当日净买入 > 近5日均量: 加速增持
   - 北向当日净买入但低于均量: 温和增持
   - 北向净卖出: 减持信号
   - 非沪深港通标的: N/A，此维度评分设为0

   5.2 **北向持续性判定**
   - 近5日北向连续增持: 强背书
   - 近5日3日增持2日减持: 波动性
   - 近5日连续减持: 资金撤离
   - 无北向数据: N/A

   5.3 **公募基金持仓变动**
   - 从 top10_floating_holders 获取公募/社保持仓变动
   - 公募基金新进或增持: 中线看好
   - 公募减持: 中线不看好
   - 社保基金增持: 极强中线信号(国家队背书)

   5.4 **北向与公募评分**
   - 北向加速增持+社保增持(+20): 最强中线背书
   - 北向持续增持+公募增持(+15): 强背书
   - 北向温和增持(+10): 有背书
   - 北向净卖出(-10): 资金撤离
   - 非沪深港通标的(0): N/A不加分不扣分
   - 公募+社保减持(-15): 中线资金不看好

   评分区间：[-15, +20]

6. **维度4：分时盘口资金分析（日内起爆信号）**

   研判分时资金强弱、分时资金背离、脉冲资金、回流资金、日内承接资金强度。日内分时持续净流入、回调无砸盘、下跌快速回流，是隔日涨停最强短线资金信号。

   6.1 **分时资金流入曲线**
   - 从 moneyflow_dc 获取分时级别资金数据
   - 分时资金持续净流入(无明显流出): 强势抢筹
   - 分时资金先流入后流出: 冲高回落型
   - 分时资金持续净流出: 弱势出逃

   6.2 **回调资金回流判定**
   - 价格回调时资金快速回流: 强承接
   - 价格回调时资金继续流出: 无承接
   - 尾盘资金回流抢筹: 隔日涨停信号

   6.3 **脉冲资金与异动**
   - 盘中突发大额扫货脉冲: 主力急迫买入
   - 无脉冲+平稳流入: 温和建仓
   - 多次脉冲+间隔回落: 试盘行为

   6.4 **分时盘口评分**
   - 分时持续净流入+强承接+尾盘抢筹(+20): 最强日内信号
   - 分时净流入+有承接(+15): 较强
   - 分时温和流入(+8): 一般
   - 分时冲高回落+资金背离(-10): 假突破
   - 分时持续流出+无承接(-15): 资金出逃

   评分区间：[-15, +20]

7. **维度5：筹码抛压资金分析（安全性过滤）**

   研判主力兑现资金、出逃资金、散户跟风资金量。排查高位资金兑现、持续性抛压、主力出货痕迹，无主力出货、资金锁仓良好的标的更容易走出连板行情。

   7.1 **主力出货痕迹**
   - 超大单卖出占比远超买入: 出货明确
   - 近3日主力资金由流入转为流出: 兑现信号
   - 高位(涨幅>30%)出现主力大额流出: 高位出货

   7.2 **散户资金行为**
   - 中单/小单净流入占比过大(>60%): 散户主导，缺乏主力
   - 中单/小单净流出: 散户恐慌出逃(如果是低位，可能是洗盘)
   - 散户跟风涌入+主力流出: 最危险组合(诱多出货)

   7.3 **筹码锁仓度**
   - 从 top10_floating_holders 判断机构持仓稳定性
   - 机构持仓未减持 + 主力净流入: 筹码锁定
   - 机构减持 + 主力流出: 筹码松动
   - 大股东无减持计划: 安全

   7.4 **抛压资金评分**
   - 无抛压+筹码锁定+主力锁仓(+20): 最安全
   - 微弱抛压+机构稳定(+15): 较安全
   - 中等抛压(+5): 有一定压力
   - 主力出货+散户涌入(-15): 诱多风险
   - 连续大额抛压+筹码松动(-20): 高危出货

   评分区间：[-20, +20]

8. **Calculate composite fund flow score**

   8.1 **原始分数汇总**
   - Composite = 主力大单 + 龙虎榜 + 北向公募 + 分时盘口 + 筹码抛压
   - raw range approximately [-90, +115]

   8.2 **维度优先级加权**
   - 主力大单权重: 1.3 (涨停核心驱动力，权重最高)
   - 龙虎榜权重: 1.2 (强势溢价信号)
   - 分时盘口权重: 1.1 (日内起爆信号)
   - 北向公募权重: 0.9 (中线背书，偏辅助)
   - 筹码抛压权重: 1.0 (安全性过滤)
   - Weighted = 1.3*主力大单 + 1.2*龙虎榜 + 1.1*分时盘口 + 0.9*北向公募 + 1.0*筹码抛压

   8.3 **归一化到0-100概率**
   - `probability = max(0, min(100, (weighted + 90) / 205 * 100))`

9. **Return result to team-leader**
   - Write result to outbox: `/srv/agent-bus/zt-fund-flow/outbox/{code}-fundflow-{timestamp}.json`
   - Include: stock code, 涨停预测, composite probability, all sub-scores, key signals, risk points, timestamp.

## Pitfalls

- 北向资金数据仅适用于沪股通/深股通股票；对于不符合条件的股票，请将子评分设为0，并明确标记“N/A”——不要惩罚。
- 单日资金流激增可能反映一次性的大宗交易，而非持续性的积累；需查看3-5天的趋势。
- 资金流数据在交易时段内可能滞后5-15分钟；解读盘中信号时需考虑此延迟。
- 大额订单数据可能会产生误导，如果一个大额订单被拆分成许多中等订单（分单行为）；尽可能与逐笔数据交叉验证。
- 请勿过度重视小盘股的北向资金；北向持仓相对于总流通股通常较小。
- 诱多出货是最危险的资金陷阱：散户涌入+主力流出，表面看似放量上涨但实质是出货，必须重点识别。
- 龙虎榜数据仅当日涨幅>7%的个股才上榜，未上榜不代表无主力，只是没有触发上榜门槛。
- 分时资金数据受盘中波动影响大，尾盘数据变化快，需在收盘后复核。
- 筹码抛压维度是安全过滤，如果该维度为负值（-20），即使其他维度得分高也应大幅降低综合评级。
- 主力大单净流入占比需要结合成交量综合判断，小盘股绝对金额小但占比可能很高。
