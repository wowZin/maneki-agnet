# ZT Sentiment Analysis

基于5维度情绪面分析框架，量化评估个股短期涨停爆发概率。仅做情绪与人气逻辑研判，不参与基本面、纯技术形态分析。

This skill runs on the **zt-sentiment** agent.

## 数据源

Tushare Pro MCP (已配置在 Hermes Agent config.yaml)

使用的 MCP 工具:
- `limit_step`: 涨停股票连板天梯 (每天连板个数晋级的股票)
- `limit_cpt_list`: 涨停最强板块统计 (每天涨停最多的概念板块)
- `ths_daily`: 同花顺概念和行业指数行情
- `ths_member`: 同花顺概念板块成分列表
- `cls_stock_shock`: 财联社个股异动 (涨停/连板/炸板/跌停池)
- `limit_list_d`: 涨跌停和炸板详细数据
- `kpl_list`: 开盘啦涨停/跌停/炸板榜单数据
- `concept`: 概念股分类表 (板块归属)
- `limit_step`: 涨停股票连板天梯 (连板天数晋级数据)

## 核心分析规则（固定执行逻辑）

- **优先级规则**：市场整体情绪 > 主线题材情绪 > 板块梯队情绪 > 个股人气情绪 > 舆情热度情绪，多重情绪共振判定为高潜力涨停标的。
- **避雷优先规则**：处于题材退潮期、高位情绪杀跌、板块资金出逃、个股人气快速下滑、炸板率走高，直接降低涨停评级，严重者直接剔除情绪标的池。
- **判定标准**：单一情绪信号仅为小幅加分，市场回暖 + 主线强势 + 板块前排 + 个股高人气 + 游资资金扎堆多维度共振，判定为高强度情绪涨停潜力股。
- **输出要求**：每次分析需明确标注【涨停预测（是/否）】+【核心情绪支撑逻辑】+【情绪面风险点】。

## Steps

1. **Receive task from team-leader**
   - Read task from inbox: `/srv/agent-bus/zt-sentiment/inbox/`
   - Extract stock code, name, board type, sector, and context.

2. **Fetch sentiment data**
   - 调用 MCP 工具 `limit_step`，获取连板天梯数据 (市场最高连板高度)
   - 调用 MCP 工具 `limit_cpt_list`，获取涨停最强板块统计 (板块共振)
   - 调用 MCP 工具 `cls_stock_shock`，获取财联社涨停池/连板池/炸板池数据
   - 调用 MCP 工具 `limit_list_d`，获取近60日涨停/连板/炸板详细记录
   - 调用 MCP 工具 `ths_daily` + `ths_member`，获取同花顺板块行情和成分
   - 调用 MCP 工具 `kpl_list`，获取开盘啦涨停/跌停/炸板榜单
   - 调用 MCP 工具 `concept`，获取概念股分类 (确定板块归属)

3. **维度1：大盘整体情绪分析（市场大环境根基）** — 权重最高

   研判全市场赚钱效应、涨跌家数、涨停家数、跌停家数、炸板率、市场连板高度、两市成交额氛围，判断整体资金做多做空情绪，大环境情绪向好是情绪涨停的基础前提。

   3.1 **市场赚钱效应**
   - 涨停家数 > 50: 极强做多情绪
   - 涨停家数 30-50: 中等偏强
   - 涨停家数 10-30: 弱势
   - 涨停家数 < 10: 冰点，几乎无赚钱效应

   3.2 **涨跌家数与炸板率**
   - 涨跌家数比 > 3:1: 大面积上涨，情绪极好
   - 炸板率 < 20%: 封板质量高，资金做多坚决
   - 炸板率 20%-40%: 分歧较大
   - 炸板率 > 40%: 封板意愿差，情绪退潮

   3.3 **市场连板高度**
   - 市场最高连板 >= 5连板: 高度打开，情绪亢奋
   - 市场最高连板 3-4连板: 中等高度
   - 市场最高连板 <= 2连板: 高度压制，情绪低迷

   3.4 **两市成交额**
   - 成交额 > 1.2万亿: 放量活跃
   - 成交额 0.8-1.2万亿: 正常
   - 成交额 < 0.8万亿: 缩量低迷

   3.5 **大盘情绪评分**
   - 涨停50+/炸板率<20%/高度>=5/放量(+25): 极强做多环境
   - 涨停30-50/炸板率20-30%/高度3-4(+18): 偏强环境
   - 涨停10-30/炸板率30-40%(+8): 一般环境
   - 涨停<10/炸板率>40%/高度<=2(-15): 退潮冰点

   评分区间：[-15, +25]

4. **维度2：主线题材情绪分析（短线情绪核心）**

   研判当下市场主流热点题材热度、题材发酵力度、政策舆情热度、题材延续性、题材轮动速度，判断题材处于启动、发酵、高潮、退潮哪个阶段，优先抓取主线强情绪标的。

   4.1 **题材热度判定**
   - 从 limit_cpt_list 获取涨停最强板块
   - 板块涨停 >= 5只: 主线级别题材
   - 板块涨停 3-4只: 支线级别
   - 板块涨停 1-2只: 个股独立行情

   4.2 **题材生命周期判定**
   - 启动期(第1-2天): 前排龙头涨停，后排未动 → 最值得介入
   - 发酵期(第2-3天): 前排继续涨停，后排开始跟风 → 龙头溢价
   - 高潮期(第3-4天): 全面涨停潮 → 注意分歧
   - 退潮期(第4天+): 龙头开板，跟风炸板 → 回避

   4.3 **题材逻辑强度**
   - 政策级利好(国家级政策/重大规划): 最强逻辑
   - 产业级利好(技术突破/供需失衡): 较强逻辑
   - 事件驱动(突发事件/短期催化): 短期逻辑
   - 无明确逻辑(纯资金炒作): 最弱

   4.4 **主线题材评分**
   - 主线级别+启动期+政策逻辑(+25): 最强题材共振
   - 主线级别+发酵期+产业逻辑(+20): 强题材
   - 支线级别+启动期(+15): 有潜力
   - 退潮期/无逻辑(-10): 题材退潮风险

   评分区间：[-10, +25]

5. **维度3：板块梯队情绪分析（跟风强弱判定）**

   研判个股所在板块内部梯队完整度、涨停梯队数量、龙头带动效应、板块内涨停扩散力度、后排跟风强度，前排标的情绪溢价远高于后排弱势跟风股。

   5.1 **梯队完整度**
   - 从 limit_step + concept 判定板块连板梯队
   - 龙头(3+连板) + 中军(2连板) + 跟风(首板): 最完整梯队
   - 有龙头无中军: 梯队断层
   - 仅有首板: 无梯队

   5.2 **个股梯队位置**
   - 龙头位置(板块最高连板): 最大情绪溢价
   - 中军位置(板块次高连板): 较高溢价
   - 跟风位置(首板): 有限溢价
   - 板块末位(最晚涨停): 溢价最低

   5.3 **龙头带动效应**
   - 龙头封板后板块跟风涨停 >= 3只: 强带动效应
   - 龙头封板后板块跟风1-2只: 弱带动
   - 龙头封板但无跟风: 独木难支

   5.4 **板块梯队评分**
   - 龙头位置+完整梯队+强带动(+25): 最强梯队效应
   - 中军位置+完整梯队(+18): 高溢价位
   - 跟风首板+有梯队支撑(+12): 有支撑
   - 无梯队+独立涨停(+5): 缺乏支撑
   - 板块末位+梯队不完整(-5): 溢价低

   评分区间：[-5, +25]

6. **维度4：个股人气资金情绪分析（起爆直接动力）**

   研判个股历史连板高度、市场辨识度、游资关注度、龙虎榜资金偏好、日内资金追捧度、市场资金记忆度，高辨识度人气个股极易走出情绪连板行情。

   6.1 **涨停基因(历史连板能力)**
   - 从 limit_list_d 获取近60日涨停记录
   - 60日内涨停 >= 5次: 强涨停基因
   - 60日内涨停 3-4次: 中等基因
   - 60日内涨停 1-2次: 弱基因
   - 封板率 > 80%: 封板能力强

   6.2 **市场辨识度**
   - 历史连板高度 >= 5连板: 极高辨识度(市场记忆强)
   - 历史连板高度 3-4连板: 高辨识度
   - 历史连板高度 2连板: 中等辨识度
   - 仅首板历史: 低辨识度

   6.3 **游资关注度**
   - 从 kpl_list 获取游资活动数据
   - 多家知名游资同时介入: 高度关注
   - 单一游资介入: 一般关注
   - 无知名游资: 低关注

   6.4 **个股人气评分**
   - 强涨停基因+高辨识度+多游资(+25): 最强人气
   - 中等基因+高辨识度(+18): 较强人气
   - 弱基因+中等辨识度(+10): 一般人气
   - 无涨停基因+低辨识度(0): 无人气基础
   - 历史炸板率高+游资出逃(-10): 负面人气

   评分区间：[-10, +25]

7. **维度5：舆情与资金风向分析（情绪催化信号）**

   研判个股相关财经舆情热度、社交平台讨论热度、利好消息传播力度、短线资金流向偏好、市场资金炒作风向，捕捉突发情绪发酵带来的涨停机会。

   7.1 **财经舆情热度**
   - 从 cls_stock_shock 获取个股异动数据
   - 个股出现在涨停池/连板池: 高舆情
   - 个股出现在异动榜: 中等舆情
   - 无异动记录: 低舆情

   7.2 **利好消息传播**
   - 有国家级政策直接利好: 最强催化
   - 有行业级利好消息: 较强催化
   - 有公司级利好(公告/订单): 一定催化
   - 无明显消息催化: 仅靠资金驱动

   7.3 **资金风向偏好**
   - 当前市场资金偏好与个股所属板块一致: 顺风
   - 资金偏好与个股板块无关: 中性
   - 资金偏好远离个股板块: 逆风

   7.4 **舆情风向评分**
   - 强舆情+强催化+资金顺风(+20): 最强催化
   - 中等舆情+有催化+资金顺风(+15): 较强催化
   - 弱舆情+无催化+中性(+5): 一般
   - 无舆情+无催化+逆风(-10): 情绪逆风

   评分区间：[-10, +20]

8. **Calculate composite sentiment score**

   8.1 **原始分数汇总**
   - Composite = 大盘情绪 + 主线题材 + 板块梯队 + 个股人气 + 舆情风向
   - raw range approximately [-50, +120]

   8.2 **维度优先级加权**
   - 大盘整体情绪权重: 1.3 (大环境根基，权重最高)
   - 主线题材情绪权重: 1.2 (短线情绪核心)
   - 板块梯队情绪权重: 1.1 (跟风强弱)
   - 个股人气情绪权重: 1.0 (直接动力)
   - 舆情风向情绪权重: 0.8 (催化信号，偏辅助)
   - Weighted = 1.3*大盘 + 1.2*题材 + 1.1*梯队 + 1.0*人气 + 0.8*舆情

   8.3 **归一化到0-100概率**
   - `probability = max(0, min(100, (weighted + 50) / 170 * 100))`

9. **Return result to team-leader**
   - Write result to outbox: `/srv/agent-bus/zt-sentiment/outbox/{code}-sentiment-{timestamp}.json`
   - Include: stock code, 涨停预测, composite probability, all sub-scores, key signals, risk points, timestamp.

## Pitfalls

- 涨停基因分析要求至少30个交易日的历史数据；对于首次公开募股（不足30天）的股票，请将其标记为“历史数据不足”，并将子评分设为0。
- 连板股（3+连板）存在急剧回调的较高风险；即使评分为正，也务必标记该风险。
- 没有明确逻辑（无明确逻辑）的板块共振通常难以持久；需区分政策驱动和投机驱动的热度。
- 龙头股识别是基于判断的；不确定时，请使用“可能龙头”而非“确认龙头”。
- 请勿将龙头股地位视为永久；如果竞争对手拥有更强的封板，一只股票可能会在单个交易时段内失去龙头股地位。
- 大盘情绪权重最高，大盘退潮期（涨停<10/炸板率>40%）无论个股多强都要大幅降级。
- 题材生命周期判定是情绪面的关键，高潮期跟风首板次日溢价极低，需特别标注风险。
- 舆情数据存在滞后性，盘中突发利好可能尚未反映在数据中，需结合实盘判断。
- 板块梯队断层（有龙头无中军）意味着跟风不足，龙头开板后板块可能快速崩溃。
