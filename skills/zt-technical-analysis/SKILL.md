# ZT Technical Analysis

基于5维度技术面分析框架，量化评估个股短期涨停爆发概率。量能分析已合并入技术面（原 zt-volume-analyst 量价分析）。

This skill runs on the **zt-technician** agent.

## 数据源

Tushare Pro MCP (已配置在 Hermes Agent config.yaml)

使用的 MCP 工具:
- `daily`: 历史日线行情 (open/high/low/close/vol/amount/pct_chg)
- `daily_basic`: 日线指标 (换手率/量比/市盈率等)
- `stk_factor_pro`: 股票技术面因子专业版 (MACD/KDJ/RSI/布林带/均线等)
- `stk_mins`: 历史分钟数据 (支持1/5/15/30/60min)
- `stk_limit`: 每日涨跌停价格 (判断封板情况)
- `limit_list_d`: 涨跌停和炸板详细数据 (封板时间/封板量/炸板次数)

## 核心分析规则（固定执行逻辑）

- **优先级规则**：量价异动爆发 > 趋势多头结构 > 筹码锁仓集中 > 关键位置突破 > 分时资金回流，多重技术逻辑叠加判定为高潜力涨停标的。
- **避雷优先规则**：出现放量破位、跌破关键均线、高位巨量滞涨、筹码松动、连续缩量阴跌等负面技术形态，直接降低涨停评级，严重者直接剔除潜力标的池。
- **判定标准**：单一技术信号仅小幅加分，放量突破 + 多头排列 + 筹码集中 + 关键支撑企稳 + 主力资金净流入多维度共振，判定为高强度技术面涨停潜力股。
- **输出要求**：每次分析需明确标注【涨停预测（是/否）】+【核心技术支撑逻辑】+【技术面风险点】。

## Steps

1. **Receive task from team-leader**
   - Read task from inbox: `/srv/agent-bus/zt-technician/inbox/`
   - Extract stock code, name, board type, and any provided context.

2. **Fetch technical data**
   - 调用 MCP 工具 `daily`，获取近60个交易日日线行情 (OHLCV)
   - 调用 MCP 工具 `daily_basic`，获取换手率、量比数据
   - 调用 MCP 工具 `stk_factor_pro`，直接获取技术面因子数据 (MACD/KDJ/RSI/均线等)
   - 调用 MCP 工具 `stk_mins`，获取当日60分钟K线 (5根) 和5分钟K线
   - 调用 MCP 工具 `stk_limit`，获取涨跌停价格
   - 调用 MCP 工具 `limit_list_d`，获取封板数据 (封板时间/封板量/炸板次数)

3. **维度1：量能结构分析（涨停核心动力）** — 权重最高

   量能是涨停的核心驱动力。重点研判当日/近3日成交量变化、量比、换手、放量缩量节奏。识别放量启动、缩量洗盘、再次放量突破的标准起爆结构；规避无量拉升、放量出货、持续缩量弱势形态。

   3.1 **成交量趋势分析**
   - 计算近5日/10日/20日平均成交量
   - 当日成交量 vs 5日均量：>2倍为显著放量，1.5-2倍为温和放量，<0.7倍为缩量
   - 量比：量比>2为明显放量，1.5-2为温和放量，<1为缩量
   - 换手率：换手率3%-8%为活跃区间，>8%为极度活跃(注意出货)，<1%为低迷

   3.2 **放量缩量节奏判定**
   - 标准起爆结构：缩量回调 → 放量突破 → 确认突破有效
   - 危险结构：高位放量滞涨(量大但不涨) → 出货信号
   - 危险结构：上涨无量配合 → 假突破信号
   - 连续3日温和放量递增 → 主力吸筹信号

   3.3 **量价配合评分**
   - 放量突破+量比>2+换手活跃(+25): 标准起爆量能
   - 温和放量+趋势向上(+18): 健康上涨量能
   - 缩量回调后放量(+15): 洗盘结束信号
   - 无量拉升/量价背离(-15): 假突破风险
   - 高位放量滞涨(-20): 出货信号

   评分区间：[-20, +25]

4. **维度2：趋势与均线分析（行情基础骨架）**

   研判个股短中长期均线排列、趋势方向、高低点抬升情况。优先判定多头排列、回踩均线企稳、趋势向上的强势标的；坚决排除均线空头、压力压制、趋势走弱的弱势标的。

   4.1 **均线排列分析**
   - 从 stk_factor_pro 获取 MA5/MA10/MA20/MA60 数据
   - 完全多头排列(MA5>MA10>MA20>MA60): 强势趋势
   - 部分多头(MA5>MA10>MA20 但 MA20<MA60): 中期趋势待确认
   - 空头排列: 弱势，直接降低评级

   4.2 **股价与均线位置**
   - 股价站上所有均线: 极强位置
   - 股价在MA5-MA20之间: 中等位置
   - 股价跌破MA20: 弱势信号
   - 回踩MA10/MA20企稳反弹: 经典买点

   4.3 **趋势与均线评分**
   - 完全多头排列+股价站上所有均线(+25): 最强趋势
   - 部分多头+回踩均线企稳(+18): 趋势形成中
   - 均线收敛+方向待选(+8): 观望
   - 空头排列(-15): 趋势弱势
   - 跌破关键均线(-20): 破位信号

   评分区间：[-20, +25]

5. **维度3：关键位置形态分析（起爆临界点）**

   研判股价所处位置：平台突破、箱体突破、新高突破、回踩支撑、W底、启明星、揉搓线、洗盘结束形态。涨停绝大多数诞生在关键压力位突破、企稳反转位置。

   5.1 **K线形态识别**
   - 看涨反转形态：锤子线、吞没形态、早晨之星、红三兵
   - 看涨持续形态：跳空缺口、光头阳线、揉搓线
   - 洗盘结束形态：缩量十字星后放量阳线、长下影线企稳
   - 看跌信号：长上影线、高位十字星、黄昏之星

   5.2 **位置与突破判定**
   - 平台/箱体放量突破: 最强起爆点
   - 前高压力位突破: 打开上涨空间
   - 回踩前期支撑企稳: 二次启动机会
   - 高位滞涨: 随时可能回调

   5.3 **位置形态评分**
   - 关键位放量突破(+25): 起爆临界点
   - 看涨K线形态+趋势配合(+18): 形态共振
   - 洗盘结束信号(+15): 即将启动
   - 无明显形态特征(+5): 中性
   - 高位滞涨/看跌形态(-15): 回调风险
   - 破位形态(-20): 止损信号

   评分区间：[-20, +25]

6. **维度4：筹码结构分析（连板持续性核心）**

   研判筹码峰位置、筹码集中度、上方套牢盘、下方获利盘。低位单峰密集、上方无压力、筹码锁定良好是连板与涨停的核心技术前提；高位筹码发散、上方重压极易冲高回落。

   6.1 **筹码集中度分析**
   - 从 stk_factor_pro 获取筹码相关因子
   - 低位单峰密集: 最优筹码结构
   - 双峰填谷(吸筹完成): 即将突破
   - 筹码发散: 分散无主力
   - 高位单峰: 主力可能出货

   6.2 **套牢盘与获利盘**
   - 上方套牢盘比例<20%: 抛压轻，涨停阻力小
   - 上方套牢盘20%-40%: 中等抛压
   - 上方套牢盘>40%: 抛压重，突破困难
   - 下方获利盘稳定(未大幅兑现): 筹码锁定好

   6.3 **筹码结构评分**
   - 低位单峰密集+套牢盘<20%(+25): 最优筹码结构
   - 双峰填谷+即将突破(+18): 吸筹完成
   - 筹码中等集中(+10): 一般
   - 筹码发散+无主力痕迹(0): 无主力
   - 高位筹码发散+上方重压(-15): 出货风险
   - 高位松动+放量换手(-20): 主力出逃

   评分区间：[-20, +25]

7. **维度5：资金与盘口分析（短线起爆信号）**

   研判主力净流入、大单占比、分时强弱、板块联动、日内承接力度、打板资金意愿。资金持续流入、日内强承接、逆势抗跌，是短线隔日涨停的最强前置信号。

   7.1 **主力资金方向**
   - 从 limit_list_d 获取封板数据(封板时间/封板量/炸板次数)
   - 早盘封板(10:00前) + 封板量大 + 0炸板: 最强盘口
   - 午盘封板 + 封板量中等: 较强
   - 尾盘封板 + 炸板多次: 偏弱，次日风险大

   7.2 **分时强弱判定**
   - 从 stk_mins 获取分时数据
   - 分时线始终在均价线上方: 极强
   - 分时线围绕均价线波动: 中等
   - 分时线长时间在均价线下方: 弱势

   7.3 **资金与盘口评分**
   - 早盘快速封板+零炸板+封板量大(+25): 最强盘口
   - 强势封板+承接有力(+18): 较强资金
   - 分时资金净流入+趋势向上(+12): 资金配合
   - 尾盘封板+多次炸板(-10): 封板不牢
   - 盘口弱势+资金流出(-15): 资金出逃

   评分区间：[-15, +25]

8. **Calculate composite technical score**

   8.1 **原始分数汇总**
   - Composite = 量能结构 + 趋势均线 + 位置形态 + 筹码结构 + 资金盘口
   - raw range approximately [-95, +125]

   8.2 **维度优先级加权**
   - 量能结构权重: 1.3 (最高权重，涨停核心动力)
   - 趋势均线权重: 1.2 (行情基础骨架)
   - 筹码结构权重: 1.1 (连板持续性)
   - 位置形态权重: 1.0 (起爆临界点)
   - 资金盘口权重: 0.9 (短线信号，日间波动大)
   - Weighted = 1.3*量能 + 1.2*趋势 + 1.1*筹码 + 1.0*位置 + 0.9*资金

   8.3 **归一化到0-100概率**
   - `probability = max(0, min(100, (weighted + 95*1.0) / (125+95)*1.0 * 100))`

9. **Return result to team-leader**
   - Write result to outbox: `/srv/agent-bus/zt-technician/outbox/{code}-technical-{timestamp}.json`
   - Include: stock code, 涨停预测, composite probability, all sub-scores, key signals, risk points, timestamp.

## Pitfalls

- Do not use stale K-line data; always fetch fresh data at analysis time.
- 量能维度权重最高，无量配合的突破一律降级处理，不可仅凭K线形态给出高分。
- 高位放量滞涨是出货经典形态，必须作为避雷信号优先处理，不可解读为主力吸筹。
- 筹码结构分析对上市不足60个交易日的次新股参考价值有限，需标记"筹码数据不充分"。
- 筹码高位发散+放量换手是主力出逃的典型特征，出现时直接判定涨停概率为"否"。
- 分时盘口数据受盘中波动影响大，仅做辅助参考，不可单独作为涨停预测依据。
- Be cautious with overbought signals (RSI > 80, KDJ J > 100) — they indicate strong momentum but also reversal risk; flag these explicitly.
- Moving average analysis is less reliable for stocks with fewer than 60 trading days of history; flag these.
- Do not analyze stocks in trading halt; check halt status before fetching data.
- 量价背离(价涨量缩)是假突破信号，必须明确标注风险。
