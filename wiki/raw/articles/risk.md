# 风险提示

## 免责声明

本系统仅供技术研究与学习使用，不构成任何投资建议。

- 涨停预测受市场环境、政策变化、黑天鹅事件等影响，准确率存在不确定性
- 历史预测表现不代表未来收益
- 系统输出仅供参考，投资决策需自行判断
- 使用本系统进行实际交易造成的损失，开发者不承担任何责任

## 技术风险

### 数据风险

- API 数据延迟可能导致分析基于过时信息
- 行情数据源故障将导致无法正常扫描
- 数据异常值可能影响评分准确性

### 模型风险

- 子 agent 的评分模型基于历史规律，市场结构变化可能导致失效
- 权重自适应存在滞后性，市场风格切换时可能调整不及时
- 多维度加权并不能消除所有误判风险

### 系统风险

- Docker 容器异常退出可能导致错过扫描窗口
- 网络故障可能导致飞书通知发送失败
- 并发分析时资源不足可能导致超时

## 风控措施

| 措施 | 说明 |
|---|---|
| 权重确认 | 权重调整需人工确认 |
| 最小/最大权重 | 任何agent权重不低于10%不高于40% |
| 保守评分 | 不确定时倾向偏低分 |
| 超时弃权 | 5分钟未返回则跳过该维度 |
| 通知重试 | 飞书通知最多重试3次 |
| 零信号静默 | 无置信度>35%的股票不发送通知 |
| 日志审计 | 所有预测和复盘记录持久化 |
