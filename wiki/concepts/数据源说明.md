# 数据源说明

## Eastmoney (东方财富) — 实时数据

通过 `push2.eastmoney.com/api/qt/clist/get` API 获取。

**用途：**
- 实时行情（涨幅、涨速、量比、成交额）
- 主力资金净流入（f62 字段）
- 分时盘口数据
- 竞价数据

**访问方式：** 通过 zdtps 动态代理，`get_proxies_dict()` 直连。

## Tushare — 非实时/历史数据

通过 `http://api.tushare.pro` API 获取。

**用途：**
- 财务数据（ROE、净利润增长）
- 历史日线
- 涨停板列表（`limit_list_d`）
- 交易日历（`trade_cal`）
- 股票基本信息（`stock_basic`）

**限制：** 调用频率 `limit_list_d` 每天 10000 次。

## 数据优先级

```
requests+代理(Eastmoney实时) > Tushare T+1兜底 > 纯CDP(向后兼容)
```

## 代理机制

使用 zdtps.com 动态代理池。代理 IP 每 ~2 分钟过期。

```python
from proxy_utils import get_proxies_dict
proxies = get_proxies_dict()
resp = requests.get(url, proxies=proxies, timeout=5)
```

**可靠性策略：** 失败时 force_refresh 换 IP 重试。
