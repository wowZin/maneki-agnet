# 扫描与推送机制

## 扫描流程

```
东财API(代理) → 涨速(f11) + 涨幅(f3) 双路各200只 → 合并去重(≈400只)
    ↓
五维度评分 → 加权Top3择优 → 总分排序 → 阈值≥35
    ↓
Top 3 推送飞书
```

## 扫描时间

| 时段 | 频率 |
|------|:----:|
| 早盘① | 9:35~9:59 每5分钟 |
| 早盘② | 10:00~10:59 每5分钟 |
| 早盘③ | 11:00~11:30 每5分钟 |
| 午盘① | 13:05~13:59 每5分钟 |
| 午盘② | 14:00~14:55 每5分钟 |

## 推送规则

- 总分 ≥ 35 的股票按分降序取前 3 只
- 推送到飞书信号群
- 推送记录保存到 `data/pushed/`

## 收盘复盘 (18:00)

- 汇总当日所有推送信号
- 比对实际涨停结果
- 计算命中率、鉴别力
- 权重AB对比（如有权重变更）
