# 评估指标说明

用于衡量评分系统效果的核心指标。

## 综合分 (Composite)

0~1 之间的综合排序质量评分。由以下指标加权合成：

| 子指标 | 权重 | 说明 |
|--------|:----:|------|
| 排名分 | 0.3 | 涨停股平均排名归一化 |
| Top20覆盖率 | **0.4** | 涨停股在Top 20中的占比 |
| 分差 | 0.15 | 涨停vs非涨停的分数差距归一化 |
| AUC | 0.15 | 排序鉴别质量 |

## AUC

Area Under the Curve。随机抽一只涨停股+一只非涨停股，AUC 是"涨停股排在前面"的概率。

- 0.5 — 纯随机，无效
- 0.6 — 弱区分
- 0.7+ — 可用
- 0.8+ — 很好
- 1.0 — 完美

## 涨停均排

所有涨停股在评分排序中的**平均名次**。越低越好。

```
信号池共1248只，按总分从高到低排
涨停股平均位置 = 559名 → 中间偏下，区分度弱
目标: < 200名
```

## 信号池覆盖率

信号池（总分≥35 的股票）中包含的涨停股占全部涨停股的比例。

## 实战推送命中率

实际推送到飞书的 Top 3 中，涨停股的占比。

## 分差

涨停股的平均分 - 非涨停股的平均分。分差越大说明评分能有效区分两类股票。
