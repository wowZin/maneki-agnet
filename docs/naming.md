# Naming

Use one stable slug everywhere.

Example:

```text
Agent slug:     hermes-seo
Container:      hermes-seo
Host data dir:  /srv/hermes-seo/data
Docs dir:       agents/hermes-seo
Key names:      hermes-seo:<purpose>
Bot name:       Hermes SEO
```

Recommended base names:

```text
hermes-orchestrator
hermes-life
hermes-seo
hermes-dev
hermes-cmo
hermes-ops
hermes-finance
```

Avoid vague names:

```text
main
agent1
bot
assistant
test
```
