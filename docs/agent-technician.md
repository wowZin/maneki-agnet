# 技术/分时面Agent V2.0 — 基础分供应器

## V2.0 漏斗角色

| 项目 | 说明 |
|------|------|
| **角色** | 基础分供应器 |
| **评分** | 第二层4因子中的2个: 分时承接力(25分) + 日K突破度(20分) = 45分 |
| **作用** | 评估形态好坏和分时强度 |
| **触发时机** | 第二层精算时（每3秒） |

> V2.0变更：技术面不再独立输出总分，而是提供第二层的2个子因子评分。
> V2.0的6维度评分逻辑保留，但仅提取分时承接力+日K突破度两个因子。

---

## 第二层因子映射

| V2.0 维度 | 映射到漏斗 | 分值 |
|-----------|-----------|:----:|
| 量能结构(35分) | → 日K突破度的量比条件 | 间接 |
| 趋势均线(25分) | → 日K突破度的均线站稳条件 | 间接 |
| 关键位置形态(12分) | → 日K突破度（突破近20日高点） | 20分 |
| 筹码结构(15分) | 暂不直接使用（用于风险排除） | - |
| 日内资金动能(8分) | → 分时承接力（VWAP关系+回撤） | 25分 |
| 板块协同(5分) | 由情绪面Agent负责 | - |

### 分时承接力 (25分)

```
IF 当前价 > VWAP AND 日内回撤 < 2% AND VWAP斜率向上
  → +25分
ELIF 当前价 > VWAP 但斜率走平
  → +15分
ELIF 跌破VWAP但3分钟内收回
  → +10分
```

### 日K突破度 (20分)

```
IF 突破近20日最高价 AND 量比>1.5
  → +20分
ELIF 突破昨日最高价（弱转强）
  → +15分
ELIF 未突破但站稳所有均线(5>10>20)
  → +10分
```

## 技术面否决 → 漏斗风控

| V2.0否决 | 映射 |
|----------|------|
| 放量破位 | 分时破位 → 踢出异动池 |
| 高位滞涨 | 攻击独特性因子扣分 |
| 筹码发散 | 降低仓位建议 |
