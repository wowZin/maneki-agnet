# 【Maneki超短线涨停追击Agent V2.0】

## 一句话定位

三层漏斗非线性决策树，盘口实盘级涨停追击引擎。

## 体系架构总览

```
[全市场5000只]
    ↓  CDP短轮询 3s
┌─────────────────────┐
│ 第一层: 极速粗筛     │ 涨幅/量比/换手/防骗
└─────────┬───────────┘
    ↓ 30-80只异动池
┌─────────────────────┐
│ 第二层: 异动精算     │ 4因子评分 → Top5
│ 板块共振(30)         │
│ 分时承接(25)         │
│ 日K突破 (20)         │
│ 攻击独特(25)         │
└─────────┬───────────┘
    ↓ 5只核心打板池
┌─────────────────────┐
│ 第三层: 盘口盯盘     │ CDP 1s轮询
│ 半路/打板条件验证    │
└─────────┬───────────┘
    ↓ 触发信号
[飞书推送]
```

## 子Agent角色（V2.0重构）

| 子Agent | 角色 | V2.0定位 | 权重 |
|---------|------|----------|:----:|
| 基本面 | 排雷器 | 盘后剔除ST/立案/解禁/巨亏 | 0%（仅排雷） |
| 情绪/板块 | 方向乘数 | 板块是否热点+方向放大/缩小 | 乘数0.6~1.2 |
| 技术/分时 | 基础分 | 日K突破+分时承接+独特性 | 第二层因子(45分) |
| 资金/盘口 | 触发器 | 盘口大单+托单+防骗 | 第三层门限 |

## 关键差异（vs V1.0加权求和）

| 维度 | V1.0 | V2.0 |
|------|------|------|
| 决策方式 | 加权求和 | 层叠漏斗+非线性 |
| 基本面 | 权重20%参与评分 | 仅排雷，权重0% |
| 情绪 | 权重20%参与评分 | 乘数0.6~1.2 |
| 技术 | 权重20%参与评分 | 基础分(45分中占20) |
| 资金 | 权重20%参与评分 | 触发器(第三层) |
| 第5维度(短线) | 权重28% | 重组进攻击独特性因子 |
| 算力要求 | 中等 | 低(2C4G可行) |

## 数据流依赖

| 组件 | 依赖 | 频率 |
|------|------|:----:|
| 粗筛引擎 | CDP port 9222 + zdtps代理 | 3s |
| 精算引擎 | Tushare(静态行情/板块) + CDP快照 | 3s |
| 盘口监控 | CDP逐笔快照 | 1s |
| 复盘 | Tushare日线 | 15:00后 |

## 工程约束

1. **数据时效性**: 所有Tushare接口均为T+1数据，盘中用CDP实时
2. **2C4G适配**: 全部采用HTTP短轮询，禁止WebSocket
3. **异步I/O**: 用`asyncio + httpx`实现，低开销轮询
4. **内存计算**: 静态数据9:15前加载，盘中零DB查询
5. **动态降频**: 涨幅<3%→10s, 3%-7%→3s, >7%→1s
6. **仓位限制**: 半路≤8%/票, 打板≤15%/票, 触发即执行
