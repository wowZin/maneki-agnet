# Maneki-Agent 涨停预测系统

## 系统概览

涨停预测系统包含4个分析Agent（基本面/技术面/资金面/情绪面）和1个聚合Agent（zt-team-leader），通过定时扫描涨速数据，分发给各Agent并行分析，最后加权聚合输出涨停置信度。

### 数据源说明（重要）

| 场景 | 数据源 | 接口 | 原因 |
|------|--------|------|------|
| 盘中扫描 | requests+代理 → 东方财富API | push2.eastmoney.com API | 实时涨速、分钟级行情，代理绕过IP封锁 |
| 盘中扫描(CDP备选) | CDP+代理 → 东方财富API | push2.eastmoney.com API | requests+代理失败时降级方式 |
| 盘中分析 | requests+代理/CDP+代理 | 同上 | 当日实时数据，Tushare盘中无数据 |
| 收盘复盘 | Tushare REST API | api.tushare.pro | 历史数据完整，收盘后更新 |

**代理IP服务**: 使用动态代理(zdtps.com)绕过东方财富IP封锁。`.env`配置`PROXY_ENABLED=true`启用，代理模块`scripts/proxy_utils.py`提供统一接口。代理IP有效期约130秒，自动刷新。

**注意：Tushare daily/bak_daily/moneyflow 等接口都是 T+1 更新，收盘后才入库。盘中调用返回的是前一天数据，分析结论会出错。盘中分析必须用代理实时数据。**

### 盘中实时数据获取映射表

| 所需数据 | Tushare(T+1) | 盘中替代方案 | 获取方式 |
|---------|-------------|------------|---------|
| 量比/换手/均线 | stk_factor_pro | requests+代理→东方财富API | push2.eastmoney.com (代理绕封锁) |
| 资金流向 | moneyflow | requests+代理→东方财富CDP实时资金 | push2.eastmoney.com (代理绕封锁) |
| 涨停家数/炸板率 | limit_list_d | requests+代理→东方财富CDP板块数据 | push2.eastmoney.com (代理绕封锁) |
| 龙虎榜 | top_list | 不可用(T+1) | 盘中跳过或用大单流向代理 |
| 北向资金 | hk_hold | 不可用(已停实时披露) | 盘中跳过 |
| 人气排名 | (无) | 同花顺/东财热榜 | akshare stock_fund_flow_individual |
| 概念板块热度 | concept_detail | 东方财富概念板块 | requests+代理 + akshare |

### 全系统股票过滤规则（所有Agent共用）

任何股票满足以下任一条件，直接排除，不进入分析：
1. ST/*ST/退市整理期
2. 上市不满60日新股
3. 创业板(30xxxx.SZ) / 科创板(688xxx.SH) / 北交所(8xxxxx.BJ)
4. 当日停牌
5. 自由流通市值 < 20亿
6. 5日均换手率 < 2%
7. 连续一字板（无法买入）

### Cron 定时任务

| 任务名 | 触发时间 | 说明 |
|--------|----------|------|
| zt-surge-scan | 工作日 9:00-15:00 每10分钟 | 盘中扫描涨速，并行分析 |
| zt-daily-review | 工作日 16:00 | 收盘复盘，计算准确率 |

使用 `delegate_task(tasks=[...])` 并行调用4个子Agent，配置 `max_concurrent_children: 10` 支持最多10个并行。

## Team Leader

你是整合汇总 Agent，不独立进行行情研判，专职接收各个独立分析 Agent 的研判结果，统一记录信息、按照预设置信度权重加权计算个股涨停综合置信分数，统筹多维度结论、合并风险点，完成最终涨停概率汇总，并晚盘结束后自动生成今日复盘总结文档。

**关键行为约束（LLM必须遵守）**：
- 推送决策：综合分>=50的股票全部推送；无>=50分时推送前5只（降级兜底）
- 复盘预测池：以**今日实际推送给用户的数据**为预测集合，而非全部扫描信号。推送记录保存在 `data/pushed/`。若无推送记录，从各时段分析结果按推送规则重建（每时段独立应用推送规则，跨时段累加去重）
- 命中率：命中率 = 推送数据中涨停数 / 推送数据总数
- 涨停数据源：必须使用 Tushare `limit_list_d` 接口获取实际涨停（`limit_list` 数据不全，禁止用于复盘）
- 报告格式：复盘同时生成 JSON（程序读取）和 Markdown（人类可读）两份报告，存放于 `data/reports/YYYYMMDD.{json,md}`
- 报告字段：推送信号数、大盘涨停数、命中涨停数、命中率、命中涨停股票详情（代码+名称）
- 飞书测试模式：`.env` 中 `FEISHU_TEST_MODE=true` 时卡片标题加"测试-"前缀

详情设计参考: [Team Leader 设计](./agent-leader.md)


## 基本面 agent (V1.0 五维度量化)

专注股票基本面涨停潜力预判的智能分析Agent，核心工作为基于标准化基本面数据，量化评估个股短期、中期涨停爆发概率，仅做基本面逻辑研判，不参与技术面、纯资金盘口炒作分析。

> ⚠️ **数据源时效性**：本文档标注的Tushare接口均为T+1数据，仅适用于收盘复盘。盘中分析必须使用实时数据或代理指标。

### 五维度评分体系
- 盈利业绩(40%): ROE + 扣非净利同比(dt_netprofit_yoy) + 营收增速(or_yoy)，优先取最新季报数据（边际变化优先）
- 题材事件(30%): 概念板块数量代理事件热度 (公告事件强度待V1.5)
- 股东筹码(15%): 股东户数环比下降=筹码集中
- 财务健康(10%): 负债率+流动比率，风险扣分
- 估值性价比(5%): PE+PB相对估值

### 财务避雷一票否决（触发任意一条 → 直接0分）
- 商誉/净资产 > 30%
- 资产负债率 > 70% 且 经营现金流连续2季度为负
- 非经常性损益占净利润 > 20%
- 主业营收占比 < 50%
- 近6个月内存在 > 流通盘10% 的大额解禁

### 非线性共振加分
- 条件A：业绩≥0.8 且 事件≥0.7 且 事件窗口t≤10日 → +15分
- 条件B：业绩≥0.7 且 股东户数下降率≥0.6 → +10分
- 同时满足取最高加分，不叠加

### 评级标准
- 高：≥75分。无一票否决项，且至少2个维度≥0.7
- 中：55-74分。单一核心逻辑≥0.8，无重大风险
- 低：35-54分。逻辑薄弱或存在中等风险
- 无：<35分 或 触发否决项。不进入交易池

详情设计参考: [基本面 agent 及策略设计](./agent-fundamental.md)

## 技术面 agent (V1.0 五维度量化)

专注股票技术面涨停潜力预判的智能分析 Agent，核心工作为基于标准化 K 线、量价、趋势、盘口、均线、筹码结构数据，量化评估个股短期、隔日涨停爆发概率。仅做纯技术面逻辑研判，不参与任何基本面、政策、消息面分析。所有输出基于客观行情数据，严格遵循固定技术规则。

> ⚠️ **数据源时效性**：本文档标注的Tushare接口均为T+1数据，仅适用于收盘复盘。盘中分析必须使用CDP实时行情数据。

### 五维度评分体系 (总分100分)
1. **量能结构 (40分)**: 量比启动[±15] + 温和放量[±15] + 洗盘起爆[+10] + 换手健康[+10] → 原始分截断至[0,40]
2. **趋势均线 (25分)**: 均线排列[+15] + 回踩企稳[+10] + 死叉扣15 + MA60向下扣10 → 原始分截断至[0,25]
3. **关键位置形态 (12分)**: 平台突破[+8] + 支撑确认[+4] + 假突破扣12 → 原始分截断至[0,12]
4. **筹码结构 (15分)**: 低位密集[+10] + 锁定良好[+5] + 高位发散扣15 → 原始分截断至[0,15]
5. **资金盘口 (8分)**: 主动买盘[+5] + 分时承接[+3] + 资金出逃扣8 → 原始分截断至[0,8]

### 一票否决规则（触发任意一条 → 直接0分）
- 放量破位：收盘跌破MA20或箱体下沿，且量比>1.8
- 高位滞涨：阶段涨幅>60%，换手>25%，长上影/实体>1.5
- 筹码高位发散：套牢盘>60%或集中度较5日前扩大>30%
- 持续缩量阴跌：连续3日量比<0.5且股价重心下移
- 资金持续出逃：近2日主力净流出且分时承接<0.4

### 综合评定
- [高] ≥75分 | [中] 55~74分 | [低] 35~54分 | [无] <35分 或触发一票否决

> ⚠️ 注意：早期版本阈值（70/50/30）已废弃，全系统统一采用75/55/35。

### 数据来源
- 盘中：东方财富CDP实时行情 (push2.eastmoney.com) — 量比/换手/均线/实时资金
- 盘后：Tushare stk_factor_pro — 历史技术指标回测

详情设计参考: [技术面 agent 及策略设计](./agent-technician.md)


## 情绪面

专注股票市场情绪涨停潜力预判的智能分析 Agent，基于五维度量化评分体系评估涨停潜力。

> ⚠️ **数据源时效性**：本文档标注的Tushare接口均为T+1数据，仅适用于收盘复盘。盘中分析必须使用CDP实时数据或akshare实时接口（如stock_fund_flow_individual）。

### 五维度量化评分 V1.0（总分100分）

| 维度 | 权重 | 核心因子 | 数据源 |
|------|------|----------|--------|
| 大盘整体情绪 | 30分 | 涨停家数、跌停控制、连板高度、炸板率 | CDP实时/akshare |
| 主线题材情绪 | 30分 | 概念数量、热门题材、题材涨停 | CDP实时/akshare |
| 板块梯队情绪 | 20分 | 板块涨停、梯队完整 | CDP实时/akshare |
| 个股人气资金 | 15分 | 换手率、人气排名、涨停记忆 | CDP实时/akshare |

> 注：涨停记忆（近20日涨停次数）盘中无法实时获取，用以下代理方案：
> - 盘中：当日是否已涨停（CDP实时）+ 近5日涨幅>20% 代理
> - 盘后：Tushare limit_list_d 统计近20日涨停次数
| 舆情风向 | 5分 | 题材丰富度、概念数量 | concept_detail（概念分类表） |

> 注：舆情热度/NLP情感分析等真正舆情数据暂无稳定免费数据源，V1.0用概念数量代理。V1.5考虑接入新闻API。

### 一票否决规则（触发任意一条 → 直接0分）

1. 市场退潮：炸板率>45% 或 昨日连板晋级成功率<40%
2. 主线崩塌：核心龙头断板跌停 或 所属题材资金连续2日净流出且涨停数下降>30%
3. 高位杀跌：市场最高连板高度连续2日下降，且高位股(≥3板)平均溢价<-2%
4. 个股情绪溃散：近5日内出现≥1次"核按钮"(收盘跌幅≤-7%且放量) 或 龙虎榜知名游资单日净卖出>3000万
5. 纯跟风弱势：所属板块仅该股独涨，无梯队扩散且人气排名跌出全市场前200

### 评级标准

- ≥75分：[高] 高强度涨停潜力
- 55-74分：[中] 中等潜力
- 35-54分：[低] 低潜力
- <35分或否决：[无]

> ⚠️ 注意：早期版本阈值（70/50/30）已废弃，全系统统一采用75/55/35。

详情设计参考: [情绪面 agent 及策略设计](./agent-sentiment.md)

## 资金面

专注股票资金面涨停潜力预判的智能分析 Agent，基于五维度量化评分体系评估涨停潜力。

> ⚠️ **数据源时效性**：本文档标注的Tushare接口均为T+1数据，仅适用于收盘复盘。盘中分析必须使用CDP实时数据。北向资金实时数据已停止披露，龙虎榜为T+1，盘中需用代理指标。

### 五维度量化评分 V2.0（总分100分）

| 维度 | 权重 | 核心因子 | 数据源 |
|------|------|----------|--------|
| 超大单主力净流入 | 35分 | 规模阈值、占比健康、持续抢筹 | CDP实时资金 |
| 龙虎榜机构游资 | 25分 | 资金合力、席位主导、结构健康 | 盘中:封板质量因子(limit_list) / 盘后:top_list, top_inst |
| 分时盘口资金 | 20分 | 净流入强度、换手率 | CDP实时资金 |
| 融资聪明资金 | 7分 | 融资余额增速、融资活跃度 | 盘中:融资余额增速(margin_detail) / 盘后:hk_hold |
| 筹码抛压锁仓 | 13分 | 锁仓度高、抛压可控、换手率递减 | CDP实时资金 |

> V2.0变更：维4权重12→7(降共线性)，维5权重8→13(强锁仓)；维2盘中从大单代理→封板质量因子；维4盘中从大单代理→融资余额增速；否决3加市场状态调节；否决4加一字板豁免。

### 一票否决规则（触发任意一条 → 直接0分）

1. 主力持续流出：当日超大单+大单净流出，且近3日累计净流出 < -0.5%流通市值
2. 席位大额撤离：龙虎榜机构/游资单日净卖出 > 净买入2倍（盘后）；盘中：大单持续净流出
3. 分时资金背离：股价拉升>3%但资金净流入为负，或15分钟滑动相关系数<-0.6（市场状态调节：成交额<8000亿或跌停>20家时阈值放宽至-0.75）
4. 纯散户博弈：主力净占比 < 10%（一字板豁免：pct_chg≈10%且开板次数=0时不触发）
5. 尾盘集中兑现：14:45后资金净流出占全天流出比例>60%

### 评级标准

- ≥75分：[高] 高强度资金驱动涨停
- 55-74分：[中] 中等潜力
- 35-54分：[低] 低潜力
- <35分或否决：[无]

> ⚠️ 注意：早期版本阈值（70/50/30）已废弃，全系统统一采用75/55/35。

详情设计参考: [资金面 agent 及策略设计](./agent-fund-flow.md)
