================================================================================
          双引擎动量-均值回归混合策略（V1.0）
================================================================================

================================================================================
一、核心逻辑
================================================================================
主引擎：自适应趋势识别（KAMA + ADX），只顺中期趋势开仓。
副引擎：均值回归精确入场（动态布林带 + RSI），仅在大方向回调末端捕捉信号。
总控层：市场广度/波动率过滤器 + 跨品种信号拥挤度监控，系统性风险或信号过载时强制降仓/空仓。

================================================================================
二、指标与参数（分资产组自适应）
================================================================================
【参数分组预设】（避免多资产一刀切）
- A股日线组：KAMA(10,2,30), RSI(3), ATR(20), BB(20,2)
- 商品期货组：KAMA(14,2,40), RSI(5), ATR(20), BB(20,2)  
- 外汇/加密组：KAMA(20,3,50), RSI(7), ATR(20), BB(20,2)
*注：以下规则以A股日线组为例，其他组按预设替换周期参数*

1. 趋势方向
   - Kaufman自适应均线 KAMA(10,2,30) 与其20周期EMA的关系
   - 收盘价与 SMA(20) 的位置关系
   - ADX(14) 判断趋势强度，+DI/-DI 判断方向

2. 入场待机（动态调整）
   - 布林带(20期,2倍标准差)，带宽 = (上轨-下轨)/中轨
   - 带宽历史百分位：当前带宽在近20日的排名(0~1)
   - RSI(3)阈值动态调整：
     若带宽百分位 < 0.3 → RSI阈值放宽至20(多)/80(空)
     否则 → 维持RSI<15(多)或>85(空)
   - VWAP偏离度作为辅助（锚定修正见下文）

3. 入场触发（计分制+动态仓位）
   观察信号产生后，下一根K线开盘时对三项条件计分（满3分）：
   【计分项-做多示例，做空镜像】
   (A) 价格验证：当前价 > 观察信号K线低点 + 0.3*ATR(14)        (1分)
   (B) 成交量放大：当前量 > 过去20根均量 * 1.1                   (1分)
   (C) 未过度溢价：(Anchor_VWAP - 开盘价) < 0.5*ATR(14)          (1分)
   入场规则：
   - 总分≥2分：市价开仓，仓位系数=得分/3（2分→67%，3分→100%）
   - 仅1分：不开仓，信号作废

4. 出场规则（条件时间止损增强）
   - 移动止损：自入场最高点回落2*ATR(20)平仓
   - 条件时间止损：持仓>15根K线 AND (ADX<20 OR 最大浮盈<0.5*ATR) 平仓
     *注：ADX≥20且浮盈达标时，即使超15根K线也继续持有*
   - 信号反转：KAMA下穿EMA 且 ADX中-DI>+DI 平多
   - 分批止盈：盈利达3*ATR(20)时平50%，剩余仓位止损移至开仓均价

5. 仓位管理（波动率倒数+拥挤度降仓）
   账户权益=1,000,000（示例）
   组合目标日波动率=1%，单品种风险上限=权益2%
   理论手数 = floor(权益*0.01 / (ATR(20)*每点价值))
   最终手数 = min(理论手数 * 入场得分/3 * 拥挤度系数, 风险上限对应手数)
   *拥挤度系数：当同时触发信号的品种数>5时，系数=0.5；否则=1.0*

================================================================================
三、买入信号生成流程（做多为例）
================================================================================
【Step 0 - 总控开关】（每日盘前评估）
1. 市场广度过滤（股票）：沪深300成分股收盘价>MA(20)比例<0.25 → 禁止开多
2. 波动率过滤（期货/外汇）：HV(20)1年百分位>90% → 暂停趋势策略开仓
3. 信号拥挤度监控：全组合同时触发观察信号的品种数>5 → 所有新开仓系数×0.5

【Step 1 - 趋势过滤】（必须全部满足）
(1) KAMA(10,2,30) > EMA(KAMA,20)
(2) Close > SMA(20)
(3) ADX(14)>20 且 +DI>-DI

【Step 2 - 回调待机】
计算带宽百分位（近20日排位）：
- 带宽百分位<0.3 → RSI(3)<20
- 否则 → RSI(3)<15
同时满足：收盘价≤布林带下轨 → 标记为“观察信号”

【Step 3 - 计分入场】（下一根K线开盘时）
A. 当前价 > 观察信号低点+0.3*ATR(14)      → 1分
B. 当前量 > 过去20均量*1.1                 → 1分
C. (Anchor_VWAP-开盘价)<0.5*ATR(14)        → 1分
总分≥2 → 入场，仓位=得分/3 × 拥挤度系数

================================================================================
四、卖出/平仓规则（做多，做空镜像）
================================================================================
1. 移动止损：多仓自入场后最高价回撤2*ATR(20)平仓
2. 条件时间止损：持仓>15根K线 AND (ADX<20 OR 最大浮盈<0.5*ATR) 平仓
3. 趋势反转：KAMA下穿其EMA 且 ADX的-DI>+DI 时平多
4. 分批止盈：浮盈达3*ATR(20)时平50%，剩余仓位止损调至开仓均价，后续按移动止损跟踪

================================================================================
五、交易成本与动态滑点校准
================================================================================
1. 中国A股（日线）
   - 印花税：卖出0.05%，佣金双向0.015%
   - 基础滑点：0.05%（小盘股0.1%）
   - 动态滑点修正：实际滑点=基础滑点 × (过去20日均量/当前成交量)^0.5
   - 综合成本模拟：0.2%~0.25% + 动态滑点

2. 国内商品期货
   - 手续费：交易所基础上浮10%
   - 基础滑点：1跳
   - 动态滑点修正：实际滑点=1跳 × max(1, (过去20日均量/当前成交量)^0.5)

3. 外汇（日线）
   - 点差：欧美0.2~0.5点，交叉盘略高
   - 佣金：ECN账户$3~$5/标准手
   - 动态滑点修正：实际滑点=基础点差 × (过去20日均量/当前成交量)^0.3

4. 回测要求
   - 常规回测使用动态滑点模型，不做过度保守惩罚
   - 压力测试保留1.5%极端滑点场景，仅作尾部风险参考

================================================================================
六、工程化与风控叠加
================================================================================
- 信号执行：所有条件基于K线收盘确认，入场用下一根K线开盘价
- VWAP锚定：24小时市场使用Anchor VWAP（锚定UTC 0点或交易所结算价），非每日重置
- 参数稳健性：各资产组参数在±50%范围内变动时，策略仍需保持正期望（平原测试）
- 组合运行：同时运行多个低相关品种（股票多因子、股指期货、国债期货、商品趋势等）
- 日风控：单日账户回撤>5%停止交易1天；周回撤>10%仓位降至半仓
- 策略退役：滚动1年夏普比率<0.5，暂停子策略，季度复评
