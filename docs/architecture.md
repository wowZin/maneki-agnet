# 架构设计 V2.0 — 三层漏斗决策引擎

## 设计原则

1. **漏斗压缩**: 全市场→30-80只→5只核心，逐层精筛
2. **非线性决策树**: 废除加权求和，改用"排雷 + 乘数 + 基础分 + 触发器"分层
3. **分离关注点**:
   - 基本面 → 仅排雷（权重0%），构建可交易基础池
   - 情绪/板块面 → 乘数（决定方向，放大或缩小）
   - 技术/分时面 → 基础分（决定形态好坏）
   - 资金/盘口面 → 触发器（决定扣扳机）
4. **盘口防骗**: 垫单扫货识别 + 撤单监控 + 封单验证
5. **算力优化**: 异步HTTP短轮询 + 动态降频（2C4G友好）

## 三层漏斗数据流

```text
[全市场股票 ~5000只]
        |
        v
┌─────────────────────────┐
│ 第一层: 全市场极速粗筛   │ 每3秒轮询
│ 条件: 涨幅[2%,9.5%]     │
│       量比>2.0          │
│       换手>3%           │
│       弱转强: 开[-2,2]  │
│       当前>4%+量比>2.5  │
└─────────┬───────────────┘
          | 输出30-80只异动池
          v
┌─────────────────────────┐
│ 第二层: 异动池精算       │ 每3秒刷新
│ 4因子评分(满分100):      │
│   板块共振度(30)         │
│   分时承接力(25)         │
│   日K突破度 (20)         │
│   攻击独特性(25)         │
│ 排序取Top5进入核心池      │
└─────────┬───────────────┘
          | 输出5只核心打板池
          v
┌─────────────────────────┐
│ 第三层: 打板池高频盯盘   │ 每1秒轮询
│ 半路买点(4-7%):          │
│   ①托单>500手持续10s    │
│   ②外盘大单占比>15%     │
│   ③垫单扫货确认         │
│ 打板买点(≥9.8%):        │
│   ①封单>流通市值0.5%    │
│   ②撤单率<20%           │
│   ③卖一剩余<100手       │
└─────────┬───────────────┘
          | 输出: 买入信号
          v
    [飞书推送 / 模拟交易]
```

## 子Agent角色重构

| 子Agent | 新角色 | 权重 | 输出 |
|---------|-------|:----:|------|
| zt-fundamental | 排雷器 | 0% | 可交易池(通过/否决) |
| zt-sentiment | 方向乘数 | 乘数(0.6~1.2) | 板块方向+情绪拐点 |
| zt-technician | 基础分 | 第二层因子 | 形态评分(0-45分) |
| zt-fund-flow | 触发器 | 第三层条件 | 盘口验证+防骗 |

## 数据源适配（2C4G）

| 层 | 数据源 | 频率 | 说明 |
|----|--------|:----:|------|
| 第一层 | CDP + HTTP短轮询 | 3s | 复用现有端口9222 |
| 第二层 | Tushare(静态) + CDP | 3s | 板块/日K/独特性因子 |
| 第三层 | CDP逐笔快照 | 1s | 盘口五档+成交明细 |

> WebSocket方案暂不采用，当前用CDP HTTP短轮询即可覆盖需求。
> 后续升级路径: CDP轮询 → SSE长连接 → 云函数+消息队列

## 调度

| 调度项 | 触发条件 | 说明 |
|--------|---------|------|
| 全市场粗筛 | 每3秒 | 9:30-15:00 |
| 异动池精算 | 粗筛后有异动 | 非空即算 |
| 打板池盯盘 | 精算后有Top5 | 每1秒 |
| 收盘复盘 | 15:00后 | 汇总信号与命中 |

## 关键风控

1. **涨速陷阱**: 3分钟涨速>5%但换手<2%或内外盘比<1.1 → 剔除
2. **分时破位**: 跌破均价线3分钟未收回 → 踢出异动池
3. **情绪熔断**: 炸板率>40%且跌停>15家 → 禁止新开仓
4. **冰点试探**: 熔断+最高连板降至2板+跌停减少 → 1/4仓位试探
5. **仓位限制**: 半路≤8%/票, 打板≤15%/票
