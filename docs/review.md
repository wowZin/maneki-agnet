# 复盘机制 V2.4

## 概述

每日 15:00 收盘后自动触发复盘流程。V2.4 新增：
- **双排序对比**: 加权总分 vs Top3择优并排评估
- **攻击独特性因子**: 单独计算该因子的鉴别力
- **情绪熔断记录**: 熔断触发次数和冰点试探结果

## 复盘流程

### 1. 结果匹配

```
对当日每条推送股票(去重):
  - 查询该股票当日最终涨跌幅
  - 判定: 涨停(hit) / 未涨停(miss)
  - 判定: 胜(win) / 负(lose) — 收盘涨幅 > 扫描时涨幅 即为胜
  - 记录: weighted_score, top3_score, actual_result, hit/miss
```

### 2. 双排序命中率对比（V2.4新增）

分别计算加权总分和Top3择优的命中率：

```
加权总分排序:
  Top5:  X/5  命中率 Y%
  Top10: X/10 命中率 Y%
  
Top3择优排序:
  Top5:  X/5  命中率 Y%
  Top10: X/10 命中率 Y%
  
结论: 当前市场下 [加权/Top3] 更有效
```

### 3. 鉴别力计算

对每个子 agent:

```
鉴别力 = avg_score(命中股票) - avg_score(未命中股票)
命中率 = 命中数 / (命中数 + 未命中数) × 100%
```

鉴别力含义:
- > 0.6: 该agent评分能有效区分涨停/非涨停
- 0.3 ~ 0.6: 有一定区分能力
- < 0.3: 区分能力弱
- < 0: 评分方向反了，需要检查

#### 新增：攻击独特性因子鉴别力（V2.4）

```
攻击独特性鉴别力 = avg(攻击分_命中) - avg(攻击分_未命中)

若该因子鉴别力持续低于其他4因子 → 考虑降权或替换
```

### 4. 熔断记录（V2.4新增）

```
当日情绪状态:
  炸板率: XX%
  跌停家数: XX
  否决6熔断触发: 是/否 (XX:XX触发, XX:XX解除)
  否决7冰点试探: 是 (标的: XXX 结果: 涨停/炸板)
```

### 5. 权重调整

```
if 鉴别力 > 0.6:    delta = +5
if 0.3 < 鉴别力 <= 0.6: delta = 0
if 0 < 鉴别力 <= 0.3:   delta = -5
if 鉴别力 <= 0:         delta = -10

新权重 = clamp(当前权重 + delta, 10, 40)
归一化: 每个权重 = 新权重 / 总和 × 100
```

### 6. 案例归档

每日保存:
- 加权排序命中最高/误判最重的 Top5
- Top3排序命中最高/误判最重的 Top5
- 攻击独特性因子表现最佳的3只股票
- 供后续模型调优参考

### 7. 报告生成与推送

- 构建飞书复盘卡片（核心指标见下方模板）
- 发送到复盘专用 webhook
- 本地存档到 reviews/ 目录

## V2.4 权重配置变更记录（2026-05-22）

### 变更原因

基于321只涨停股30天回测数据和5/22盘中40只实时数据：
- 旧权重给资金面最高权(1.5→1.8)，但资金面70%股票得0分，高权被稀释
- 基本面区分度最强(P75=62)但权重仅0.7，高分贡献被低估
- 导致加权总分无法拉开，推送阈值50分几乎无人触及

### 变更内容

| 维度 | 旧权重 | 新权重 | 变更逻辑 |
|------|:--:|:--:|------|
| 基本面 | 0.7 | **1.5** | 91%有效分，区分度最强 → 给最高权 |
| 技术面 | 0.4 | **1.0** | 73%有效分，中位区分度 |
| 资金面 | 1.5 | **0.5** | 仅30%非零 → 降为加分项，不稀释总分 |
| 情绪面 | 1.2 | **1.2** | 95%有效分，维持 |
| 短线博弈 | 1.4 | **1.5** | 100%有效分，微调 |

- 加权分母: 5.0 → 5.7 (加权平均)
- 推送阈值: 50 → 35 (对应新权重下P90位置)
- 置信度: 高80→60, 中高65→45, 中等50→35, 低49→34
