# 复盘引擎 V2.0（漏斗架构适配）

## 复盘核心指标

### 漏斗各层效率

| 指标 | 计算方式 | 健康值 |
|------|---------|:------:|
| 第一层覆盖率 | 当日涨停股中被粗筛捕获比例 | > 80% |
| 第二层命中率 | Top5中最终涨停比例 | > 40% |
| 第三层触发率 | 满足盘口条件的信号占Top5比例 | > 60% |
| 假阳性率 | 触发但炸板 / 总触发 | < 30% |
| 防骗拦截率 | 盘口防骗剔除的危险信号 / 总信号 | > 20%为佳 |

### 4因子鉴别力

```
鉴别力(dim) = avg_score(命中涨停) - avg_score(未命中)

热力图:
          本周  上周  变化
板块共振  XX    XX    +X
分时承接  XX    XX    -X
日K突破   XX    XX    +X
攻击独特  XX    XX     X
```

- 鉴别力 > 10: 有效因子
- 鉴别力 5~10: 弱有效
- 鉴别力 < 5: 需调整或替换
