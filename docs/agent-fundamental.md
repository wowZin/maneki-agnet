一、 策略定位与核心交易哲学
================================================================
【Agent定位】
专注股票基本面涨停潜力预判的智能分析引擎。仅做基本面逻辑量化，不涉技术面/纯资金盘口博弈。

> ⚠️ 数据源时效性：本版Tushare等接口均为T+1收盘复盘数据。盘中分析必须使用实时数据或代理指标。
> 业绩数据优先取最新季报（边际变化优先），禁用滞后年报。

【量化哲学】
1. 涨停本质：基本面定方向与持续性，事件/资金/情绪定触发时点。
2. 因子导向：短线涨停因子聚焦边际变化与预期差（环比、加速、事件时效、困境反转），弱化静态绝对值。
3. 周期解耦：服务短线（1-20交易日），长期价值因子不纳入当期打分。
4. 抢跑防御：A股存在严重信息不对称，所有利好信号必须叠加“预期兑现度”负向修正，防范“见光死”。

二、 五维度量化因子体系
================================================================
（设计原则：行业内市值中性 + 高频化筹码 + 估值逻辑重建 + 见光死衰减）

1. 盈利业绩维度（权重：40%）
核心因子：
- SUE_扣非（标准化盈利超预期，分析师一致预期基准）
- Profit_Accel（扣非增速二阶导）
- NonRecurring_Ratio（非经常性损益占比，配合重组背景判定）
量化逻辑：
- 以券商一致预期或历史滚动QoQ为基准，正交化行业周期属性。SUE>1.5σ视为强爆发。
- 辅助因子：分析师盈利预测上调幅度（FY1 EPS Revision），不单独打分，仅用于增强SUE置信度及计算见光死惩罚。

2. 题材事件维度（权重：30%）
核心因子：
- Event_Strength（事件强度代理：国家级/供需缺口类系数1.5，普通订单类0.8）
- Time_Decay（指数衰减exp(-0.3*t)，加快衰减速度以契合短线“一波流”特征）
- Industry_Juice（行业景气弹性）
量化逻辑：
- 事件窗口硬约束 1≤t≤20，超30日自动置零。
- 协同变量：事件发生后次日高开幅度与集合竞价量能作为确认信号，提高事件因子IC。

3. 股东筹码维度（权重：15%）
核心因子：
- SH_Concentration_Filter：股东户数环比下降率降级为底仓过滤项（环比下降≥5%视为有效集中，仅用于排除极度分散个股，不直接加分）。
- Block_Trade_Premium：近20日大宗交易溢价成交比例（溢价成交占比>50% → +3分，机构抢筹信号）。
- Lockup_Impact：未来30日解禁市值/流通市值 < 2% → +2分（筹码供给安全）。
- Inst_Net_Change：机构/北向/社保季净增持占比（严格按披露日对齐，剔除被动指数基金变动）。
- Insider_Action：回购/增持强度。

4. 财务健康维度（权重：10%）
核心因子：
- CFO_Coverage（经营现金流覆盖度）
- Goodwill_Ratio（商誉/净资产，增加行业豁免：医药/电子行业商誉>30%但ROE>10%不否决）
- MainBiz_Ratio（主业营收占比）
- RD_Capitalization（研发费用资本化率，新增监控指标）
逻辑调整：
- 本维度用于一票否决与风险扣分，否决规则进行行业与背景豁免处理（见第三章避雷清单）。

5. 估值性价比维度（权重：5%）
核心因子（彻底重构，废除低PE分位加分）：
- EPS_Revision_1M（近1个月分析师盈利预测上调幅度，百分制映射）
- PEG_Forward（远期PEG，<1.5视为有估值重塑空间）
- 困境反转特殊加分：最新单季扣非净利润>0且去年同期单季<0，或存在资产重组公告 → 加5分（独立于权重，加入总分共振）。
逻辑调整：
- 此维度不再扣分，仅作为加分项，权重5%全部用于捕捉估值重塑动能。

[因子标准化流程]
原始值 → 申万二级行业内排名 → Z-Score → Sigmoid(0.5×Z)映射至[0,1]区间 → 加权求和。
**关键约束：市值中性化仅在申万二级行业内部进行，废除全市场市值正交回归，保留行业内小市值弹性。**

三、 打分模型、权重与共振规则
================================================================
【1. 基础打分公式】
Base_Score = 40×F_业绩 + 30×F_事件 + 15×F_筹码 + 10×F_财务 + 5×F_估值重塑
（注：F_x 为各维度标准化后的[0,1]得分。F_财务初始为1.0，触发避雷规则直接扣减或置零。）

【1.5 总分截断规则】
Composite_Score = min(100, Base_Score + ResonanceBonus - PreRun_Penalty)

【2. 非线性共振加分】
- 条件A：F_业绩≥0.8 且 F_事件≥0.7 且 事件窗口t≤10 → +15分
- 条件B：F_筹码≥0.8 且 F_估值重塑≥0.6 → +10分
- 规则：同时满足两项时取最高加分，不叠加。

【3. 预期兑现度（见光死）惩罚机制】
计算 Price-in_Ratio：
  涨幅_pct = 公告前20日累计涨幅 × 100 （如30%记为30）
  上调_pct = 分析师盈利预测上调幅度 × 100 （如10%记为10）
  Price-in_Ratio = 涨幅_pct / (上调_pct + 5) （分母+5防止除零并设定基础阈值）
惩罚触发：
  IF Price-in_Ratio > 2.5 且 (F_业绩>0.6 或 F_事件>0.7)：
    → F_事件因子值乘以0.5衰减系数（事件贡献腰斩）。
  IF Price-in_Ratio > 3.5：
    → 同时将F_业绩因子值乘以0.7衰减。

【4. 财务避雷一票否决清单（含行业豁免与联动）】
触发任意一条，Composite_Score = 0，直接移出潜力池：
- 商誉/净资产 > 30% 且 ROE < 10% （医药、电子行业ROE>10%暂免）。
- 资产负债率 > 70% 且 经营现金流连续2季度为负 （房地产、建筑、非银金融、公用事业行业豁免资产负债率否决，仅看现金流）。
- 扣非净利润连续3年亏损，且无重组/卖资产公告 （注：若触发“单季困境反转”加分，则豁免此条否决；单一季度非经常性损益暴增不否决）。
- 主业营收占比 < 50%。
- 近6个月内存在 > 流通盘10% 的大额解禁，且无大宗交易承接迹象。
- 研发费用资本化率 > 30% 且 经营现金流净额 < 净利润 × 0.5 （新增：利润质量存疑直接否决）。

【5. 涨停潜力等级判定阈值】
- 高：≥75分。无一票否决项，且至少2个维度≥0.7
- 中：55-74分。单一核心逻辑≥0.8，无重大风险
- 低：35-54分。逻辑薄弱或存在中等风险
- 无：<35分 或 触发否决项。不进入交易池。

四、 数据工程与回测硬约束
================================================================
【Point-in-Time (PIT) 对齐与盘后公告偏移】
所有财报、股东、公告数据严格以 announce_date 为基准。
**强制约束**：若公告发布时间在当日15:00之后（盘后公告），信号生效日必须强制递延至 announce_date 的下一个交易日（T+1），杜绝未来函数。

【横截面中性化】
每日对因子做申万二级行业虚拟变量回归，取残差参与打分。废除全市场市值中性化，改为在行业内进行市值分档比较，保留小盘股弹性。

【可交易性过滤】
自由流通市值 ≥ 20亿；5日均换手率 ≥ 2%；非ST/*ST；非连续一字板状态。

【回测协议（涨停板成交幻觉消除）】
- 若信号触发日开盘涨幅 > 7% 或 触及一字涨停 → 当日禁止成交，强制递延至次日开盘价（Next_Open）成交。
- 正常开盘允许成交，滑点0.2%。
- 计算Rank IC、IC_IR、5分组收益单调性。
- 验证底线：Rank IC > 0.05，IC_IR > 0.5，长短线收益单调。

【伪代码示例1：业绩SUE因子计算（PIT对齐+行业中性+去极值）】
def calc_sue_factor(df_pit):
    # df_pit 包含: stock, announce_date, eps_actual, eps_forecast, industry
    # 1. 绝对中位差(MAD)去极值，防止分析师乌龙指污染
    median_fc = df_pit.groupby('industry')['eps_forecast'].transform('median')
    mad = df_pit.groupby('industry')['eps_forecast'].transform(lambda x: (x - x.median()).abs().median())
    df_pit = df_pit[(df_pit['eps_forecast'] - median_fc).abs() < 3 * mad]
    
    # 2. 计算SUE
    df_pit['surprise'] = df_pit['eps_actual'] - df_pit['eps_forecast']
    df_pit['sue'] = df_pit.groupby('industry')['surprise'].transform(
        lambda x: (x - x.mean()) / x.std()
    )
    return sigmoid(0.5 * df_pit['sue'])

def realistic_fill(row):
    # 开盘涨幅>7%或一字板，强制递延至次日开盘
    if row['open_pct'] > 0.07 or row['is_limit_up_at_open']:
        return row['next_open']  
    else:
        return row['close'] * (1 + slippage)

【伪代码示例3：见光死动态衰减（量纲修正版）】
def apply_price_in_penalty(row):
    pre_return_pct = row['pre_return_20d'] * 100
    revision_pct = abs(row['eps_revision_1m']) * 100
    price_in_ratio = pre_return_pct / (revision_pct + 5)
    
    f_event = row['f_event']
    f_earnings = row['f_earnings']
    
    if price_in_ratio > 2.5:
        f_event *= 0.5
    if price_in_ratio > 3.5:
        f_earnings *= 0.7
        
    return f_event, f_earnings
