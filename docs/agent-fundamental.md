# 基本面Agent V2.0 — 降级为纯排雷器

## V2.0 漏斗角色

| 项目 | 说明 |
|------|------|
| **角色** | 排雷器 |
| **权重** | 0%（不参与评分） |
| **作用** | 盘后构建"可交易基础池"，剔除不合格标的 |
| **触发时机** | 每日开盘前（9:15前完成） |

> V2.0变更：基本面不再参与加权评分，仅做一票否决过滤。
> 其V1.5的五维度评分逻辑保留，仅在排雷模式下输出"通过/否决"。

## 排雷清单（保留V1.5否决规则）

触发任意一条 → 移出可交易基础池：

1. ST/*ST股票
2. 存在立案调查公告
3. 近6个月>流通盘10%大额解禁且无大宗承接迹象
4. 扣非净利润连续3年亏损（困境反转豁免：最新季扣非>0且去年同期<0）
5. 商誉/净资产>30%且ROE<10%（医药/电子行业ROE>10%暂免）
6. 资产负债率>70%且经营现金流连续2季度为负（地产/建筑/非银/公用行业豁免）
7. 主业营收占比<50%
8. 研发费用资本化率>30%且经营现金流<净利润×0.5
