# 评分体系 V2.0 — 漏斗评分

## 总体框架

废除 V1.0 的多维加权求和，改为三层漏斗非线性评分：

```
基本面排雷 (0分/否决) → 板块乘数 (0.6~1.2) → 基础分 (0~45) → 盘口触发 (门限)
```

最终不是算出一个总分排序，而是**层层过滤**：不满足当前层条件，直接淘汰，不进下一层。

---

## 第一层: 粗筛条件（硬门限）

| 条件 | 半路/打板 | 弱转强 |
|------|-----------|--------|
| 涨幅 | [2%, 9.5%] | 开盘[-2%,2%]→当前>4% |
| 量比 | > 2.0 | > 2.5 |
| 换手 | > 3% | > 3% |
| 涨速防骗 | 涨速>5%但换手<2%或内外盘比<1.1→剔除 | 同左 |

全量通过 → 进入第二层（约30-80只）

---

## 第二层: 异动池精算（满分100分）

### 1. 板块共振度 (30分)

| 条件 | 得分 |
|------|:----:|
| 板块涨幅全市场前5 + ≥2只涨停 | +30 |
| 板块涨幅前10 + 1只涨停 | +20 |
| 其余 | 0 |

> 新增：板块内涨停股同概念（如光刻机）→ 系数1.2；热点散乱→0.8

### 2. 分时承接力 (25分)

| 条件 | 得分 |
|------|:----:|
| 当前价>VWAP + 日内回撤<2% + VWAP斜率向上 | +25 |
| 仅价>VWAP但斜率走平 | +15 |
| 跌破VWAP但3分钟内收回 | +10 |

### 3. 日K突破度 (20分)

| 条件 | 得分 |
|------|:----:|
| 突破近20日最高价 + 量比>1.5 | +20 |
| 突破昨日最高价（弱转强） | +15 |
| 未突破但站稳所有均线 | +10 |

### 4. 个股攻击独特性 (25分) — 新增

| 条件 | 得分 |
|------|:----:|
| 近20日有过涨停 + 次日高开率>50% | +10 |
| 近10日最大涨幅>7% | +10 |
| 昨日涨停+今日弱转强（叠加） | +5 |

> 总分排序取Top5 → 进入第三层核心打板池

---

## 第三层: 盘口触发条件（门限）

### 半路买点（涨幅4%~7%）

**必须同时满足三项：**
1. 买一/买二连续托单 >500手 持续10秒以上
2. 主动外盘大单(>100手) 近1分钟净买入占比>15%
3. 连续3笔以上主动吃掉卖一压单

### 打板买点（涨幅≥9.8%）

**必须同时满足三项：**
1. 封单金额 > 流通市值0.5% 或 >3000万
2. 封死后3秒内累计撤单比例 < 20%
3. 封板后5秒内卖一剩余压单 < 100手

---

## 情绪退潮熔断

| 状态 | 条件 | 操作 |
|------|------|------|
| 熔断 | 炸板率>40% + 跌停>15家 | 禁止新开仓 |
| 冰点试探 | 熔断 + 最高连板降至2板 + 跌停减少 | 1/4仓位试探"破冰板" |

---

## 仓位规则

| 模式 | 单票上限 | 说明 |
|------|:--------:|------|
| 半路买入 | 8% | 半路浮盈>3%才可加仓打板 |
| 打板买入 | 15% | 直接打板 |
