# 资金/盘口面Agent V2.0 — 触发器+防骗

## V2.0 漏斗角色

| 项目 | 说明 |
|------|------|
| **角色** | 触发器（决定扣扳机） |
| **位置** | 第三层打板池高频盯盘 |
| **作用** | 盘口大单验证 + 托单监控 + 垫单扫货识别 + 封单撤单监控 |
| **触发时机** | 第三层（每1秒轮询） |

> V2.0变更：资金面从评分维度降级为第三层盘口触发器，不再参与评分。
> V2.3的资金面否决逻辑仅保留在排雷阶段使用。

## 半路买点（涨幅4%~7%）

**必须同时满足以下三项：**

### 条件① 连续托单

```
买一/买二队列中，单笔>500手的托单持续10秒以上未撤单
（CDP扫描间隔1秒，连续10次快照均存在该托单视为通过）
```

### 条件② 主动外盘大单

```
近1分钟内，主动外盘大单(>100手)的净买入占比 > 15%
净买入占比 = (主动买-主动卖) / (主动买+主动卖) × 100%
```

### 条件③ 垫单扫货

```
连续3笔以上主动吃掉卖一压单，且卖一价位被向上打穿
（CDP快照对比上一次快照：卖一价格上升）
```

## 打板买点（涨幅≥9.8%）

**必须同时满足以下三项：**

### 条件A 封单验证

```
封单金额 > 流通市值 × 0.5%  或  封单金额 > 3000万
```

### 条件B 撤单监控

```
封死后3秒内，累计撤单比例 < 20%
撤单比例 = 累计撤单量 / 封单总量 × 100%
若快速撤单（比例≥20%）→ 放弃
```

### 条件C 卖一压力

```
封板后5秒内，卖一剩余压单 < 100手
（表明抛压极轻，封板稳固）
```

## 盘口防骗机制

| 骗线类型 | 识别方法 | 处理 |
|----------|---------|------|
| 虚假托单 | 托单持续<10秒即撤 | 不满足条件① |
| 假外盘 | 外盘大单但内外盘比<1.1 | 第一层涨速防骗过滤 |
| 撤单钓鱼 | 封单大但撤单比例高 | 条件B拦截 |
| 抛压伪装 | 封板后卖一压单大 | 条件C拦截 |
