# 工作流 V2.0 — 三层漏斗流程

## 盘中扫描流程

### 阶段 1: 全市场粗筛 (每3秒)

```text
动作: CDP HTTP短轮询全市场涨速/涨幅
条件:
  半路/打板: 涨幅[4%,9.5%] + 量比>2.0 + 换手>3%
  弱转强:   开盘[-2%,2%] + 当前>4% + 量比>2.5
防骗过滤:
  涨速>5% 但 换手<2% 或 内外盘比<1.1 → 剔除
输出: 异动池 30-80只
```

### 阶段 2: 异动池精算 (每3秒)

```text
输入: 异动池
动作: 对每只计算4因子:
  1. 板块共振度 (30分) — 板块涨幅排名+涨停家数
  2. 分时承接力 (25分) — VWAP关系+斜率+回撤
  3. 日K突破度  (20分) — 近20日高点/昨日高点突破
  4. 攻击独特性 (25分) — 历史涨停+溢价+基因
排序: 按总分取Top5
输出: 核心打板池 5只
```

### 阶段 3: 打板池高频盯盘 (每1秒)

```text
输入: 核心打板池 5只
动作: 高频检查盘口条件:
  半路条件(4-7%): 托单+外盘大单+垫单扫货
  打板条件(≥9.8%): 封单+撤单+卖一余量
输出: 买入信号
```

### 推送

```text
条件: 触发买入信号 (半路或打板)
动作: 飞书消息推送
格式: 买入标的 + 模式 + 当前价格 + 仓位建议
```

## 风控子流程

### 分时破位剔除

```text
IF 异动池股票跌破分时均价线 AND 3分钟未收回
  → 从异动池踢出
ELIF 收回但量能萎缩
  → 降权至第二层，不再进入第三层监控
```

### 情绪熔断

```text
IF 炸板率>40% AND 跌停>15家
  → 禁止新开仓
  IF 最高连板已降至2板 AND 跌停数较前5分钟减少
    → 允许1/4仓位试探"破冰板"
    → 必须是该板块内最强逆势股
```

## 收盘复盘流程

### Step 1: 信号汇总

```text
动作: 汇总当日所有触发信号
分类: 半路触发 / 打板触发 / 未触发
标记: 实际是否涨停 = hit/miss
```

### Step 2: 漏斗效果评估

```text
指标:
  第一层命中率: 异动池中最终涨停的比例
  第二层命中率: Top5中最终涨停的比例
  第三层触发率: 满足条件的信号占Top5的比例
  
新增指标:
  假阳性率: 触发但炸板 / 总触发
  防骗拦截率: 通过盘口防骗排查掉的危险信号
  冰点成功率: 冰点试探信号的后续表现
```

### Step 3: 因子鉴别力

```text
对第二层的4个因子分别计算鉴别力:
  板块共振度: avg_score(命中) - avg_score(未命中)
  分时承接力: 同上
  日K突破度: 同上  
  攻击独特性: 同上
  
鉴别力<0 → 该因子需调整或替换
```

### Step 4: 生成复盘报告

```text
内容:
1. 今日统计 (粗筛量/精算量/触发量/命中量)
2. 漏斗各层命中率
3. 4因子鉴别力排名
4. 盘口防骗拦截案例
5. 情绪熔断触发记录
6. 精选案例 (最优/最差信号)
```
