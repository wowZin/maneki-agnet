# 情绪/板块面Agent V2.0 — 方向乘数

## V2.0 漏斗角色

| 项目 | 说明 |
|------|------|
| **角色** | 方向乘数（决定是否值得参与） |
| **权重** | 乘数 0.6~1.2 |
| **作用** | 判定板块是否为当日热点，决定方向和放大/缩小系数 |
| **触发时机** | 第二层精算时（每3秒） |

> V2.0变更：情绪面不再独立输出评分，而是作为乘数乘以基础分。
> 板块非热点 → 乘数0.6（不参与）；板块热点明确 → 乘数1.0；
> 板块内部概念一致 → 乘数1.2。

---

## 乘数计算

### 板块热度判定

```
IF 板块涨幅全市场前5 AND ≥2只涨停
  → 乘数 1.2（热点一致，加强）
  IF 板块内涨停股均为同一概念（如光刻机）
    → 加强系数再 ×1.0 (合计1.2)
  ELSE 热点散乱（不同概念）
    → 加强系数 ×0.8 (合计1.0)
ELIF 板块涨幅前10 AND ≥1只涨停
  → 乘数 1.0（正常热点）
ELSE
  → 乘数 0.6（非热点，放弃）
```

### 融合情绪面V2.3的否决逻辑

情绪面的否决规则保留，作为熔断判断的一部分：

| 否决 | 映射到漏斗 |
|------|-----------|
| 否决1 市场退潮(炸板率>45%) | 情绪熔断 → 禁止新开仓 |
| 否决3 高位杀跌(连板连续下降) | 乘数降至0.6 |
| 否决5 纯跟风弱势(人气低) | 攻击独特性因子减分 |
